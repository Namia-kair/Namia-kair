## 📈 My Growth Journey (我的成长轨迹)

我的技术成长，是一部从“使用系统”到“定制系统”的进化史。

### 阶段一：追求极致的“裁剪师”
**关键词：Ubuntu 精简、系统优化**
- 最初，我沉迷于将庞大的系统变得轻量高效。
- 实践：在 PC 与小主机上反复打磨 Ubuntu 系统，探索在保留核心功能的前提下最小化系统体积的方案。
- 收获：掌握了系统底层架构、分区策略与备份恢复的核心逻辑。

### 阶段二：固件世界的“建筑师”
**关键词：OpenWrt/LEDE、源码编译**
- 进阶到嵌入式领域，开始挑战从 0 到 1 编译 OpenWrt 固件。
- 实践：解决依赖地狱、定制 `.config`、移植 LuCI 插件、处理交叉编译错误。
- 收获：读懂了日志里的每一个报错，理解了固件从源码到烧录的完整生命周期。

### 阶段三：架构探索的“拓荒者”
**关键词：ARM 虚拟化、固件升级**
- 目前，我正在探索更复杂的边缘计算场景。
- 实践：在 ARM 架构中部署 QEMU/KVM，研究嵌入式硬件的固件升级方案。
- 收获：打破了硬件架构的壁垒，具备了跨平台开发与排错的能力。
